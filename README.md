# Simple Note Chrome Extension v2

## Description

This Chrome extension provides a secure and convenient way to take and manage encrypted notes directly in your browser. With a clean, user-friendly interface, it allows you to create multiple tabs of notes, each encrypted and stored locally for privacy.

## Features

1. **Side Panel Interface**: Opens as a Chrome side panel for easy access alongside your browsing.
2. **Multi-Tab Support**: Create and manage multiple tabs of notes with drag-and-drop reordering.
3. **Local Encryption**: All notes are encrypted before being saved locally, ensuring your data remains private.
4. **Persistent Storage**: Your notes remain available even after restarting Chrome.
5. **Easy Tab Management**:
   - **+** button: Add new tabs
   - **-** button: Delete tabs (requires two clicks for confirmation)
   - Drag tabs to reorder them
6. **Copy/Paste Between Tabs**:
   - **^** button: Copy current tab's content
   - **v** button: Paste copied content to current tab
7. **Export Function**: **>** button to export the current tab's content to a text file.
8. **Toggle Panel**: Click the extension icon to open/close the side panel.

## How to Use

1. Click on the extension icon to open the side panel (click again to close it).
2. Type your notes in the text area. Your notes are automatically saved as you type.
3. Your notes are automatically encrypted and saved locally.
4. Use the control buttons at the top:
   - **+**: Add a new tab
   - **-**: Delete current tab (click twice within 3 seconds to confirm)
   - **^**: Copy current tab's content to clipboard
   - **v**: Paste previously copied content to current tab
   - **>**: Export current tab to a text file
5. Tabs are numbered sequentially (1, 2, 3, ...) and displayed vertically on the left.
6. Drag and drop tabs to reorder them.

## Privacy and Security

- All notes are encrypted using AES encryption before being stored locally.
- Your data never leaves your browser, ensuring maximum privacy.
